# AI 个性化学习路径规划师

这是一个基于百度千帆大模型（ERNIE-Bot-turbo）开发的命令行工具，可以根据你的学习主题、当前水平和时间预算，自动生成分阶段的详细学习计划。

## 功能特点
- 🤖 基于 AI 大模型生成专业学习方案
- 📊 分阶段规划（基础、进阶、实战）
- 📋 包含核心知识点和练习方式
- 🔒 密钥安全配置（不泄露 AK/SK）
